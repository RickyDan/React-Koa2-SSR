# React 同构项目 demo

## React + Webpack + Koa2 + Mongodb

编辑器建议使用 vscode

```
1. 安装 `Node.js[必须]` `MongoDB[必须]`
2. 启动 MongoDB `$ mongod -f /usr/local/etc/mongod.conf`
3. `$ npm install` 安装项目 的依赖包
4. `cp config.default.js config.js` 请根据需要修改配置文件
6. `$ npm run dev`
7. visit `http://localhost:3000`
8. done!
```
